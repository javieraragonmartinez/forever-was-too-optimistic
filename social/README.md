# Publication cards

Four vector cards designed for transparent communication of the independent preprint:

1. `01-preprint.svg` — general announcement.
2. `02-colision.svg` — exact rational collision.
3. `03-local-global.svg` — local versus global distinction.
4. `04-reproducibilidad.svg` — reproducibility stack.

All cards state or imply the work's preprint status and should be published with a link to the repository and an explicit note that peer review is pending.
